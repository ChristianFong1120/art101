Lab 3: File Structures

Subject

The subject of this lab is to gain a stronger understanding of file structures and the way computers organize files. This is achieved by linking multiple html files in different locations to each other through links and the "a href" tag.

What is a File Structure?

One can imagine a file structure as a tree of sorts, where the each folder is a different branch amongst the same tree. The root of the tree is the root directory, and going up the tree is synonymous with going down the file structure. Each branch can be imagined as a folder and each subranch is a subfolder.
